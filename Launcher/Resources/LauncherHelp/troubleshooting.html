Launcher Help

Global hotkeys can interfere with other applications. Generally, the only side-effect is the current running application open, having the same hotkey, will respond when you initiate the global hotkey. This produces two events; the application open responds and the launching of the global hotkey is launched.

You can delete the stored menuitems from your Application Support folder and relaunch Launcher to remove all items at once. But, that is not how the application is intended to be used. Normally, you will have only a few favorites that can be modified through the interface.
