Launcher Help

Launcher is a light-weight application without a main window. It launches and registers itself on the main menubar and waits for user interactions.

When you add items Launcher collects the new menuitems and builds listeners that wait for hotkey events. Those listeners are removed and added when the app exits or launches. If you remove an item, Launcher removes all the listeners and then restores only the remaining active menuitems.

When Launcher exits, it automatically saves its menuitems to a binary file in your home folder's Library folder in the Application Support folder. Here is a path to follow.

~/Library/Application Support/Launcher/menuItems.bin

This means, if you wanted to, you could copy this file and add it to other Mac OS X machines, provided the menuitems original links are also present in the same place on the new machine.

Your preferences, containing whether Launcher launches at startup is located in:

~/Library/Preferences/com.crankysoft.Launcher.plist

That's about it. Enjoy.
